# 安全政策

## 报告漏洞

如果您发现安全漏洞，请按照以下步骤报告：

1. **不要**在公开 Issue 中报告安全漏洞
2. 发送邮件至 security@hanbao233.fun
3. 在邮件中描述漏洞的详细信息：
   - 漏洞类型
   - 复现步骤
   - 影响范围
   - 可能的修复建议

## 响应时间

- 收到报告后 48 小时内确认收到
- 7 天内提供初步评估
- 30 天内发布修复补丁

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| 1.0.x | :white_check_mark: |
| < 1.0 | :x: |

## 安全建议

- 请及时更新到最新版本
- 修改默认管理员密码
- 定期备份数据库
- 在生产环境中使用 HTTPS
